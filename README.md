# 📊 Sales Summary Dashboard – Python + SQLite + Matplotlib

This project uses a small SQLite database to analyze sales data and visualize it using bar, pie, and line charts.

## ✅ Features

- Uses SQLite for querying sales data
- Generates 4 charts:
  - Revenue by Product
  - Region-wise Revenue
  - Customer Type Revenue
  - Daily Sales Trend
- Simple and beginner-friendly!

## 🛠️ Tools Used
- Python
- SQLite (`sqlite3`)
- pandas
- matplotlib

## 📌 How to Run
1. Make sure Python is installed
2. Run the script:
```bash
python sales_summary.py
